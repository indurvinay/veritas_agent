[1:00 — ARCHITECTURE]
(show assets/architecture_diagram.png)
Let's walk through the architecture. The frontend is built with React 18, Vite, and Three.js, producing a beautiful 3D grid line and starry background. The voice interface communicates with browser-native speech recognition. 

The frontend sends messages to our Node.js and Express backend. The backend manages direct Google OAuth2 flows securely, saving tokens locally. When a request is received, the backend executes parallel searches across the Gmail, Calendar, and Drive APIs. The Gemini 2.5 Flash model handles natural language intent parsing and email drafting, while a local LowDB database acts as our session audit store.

[3:00 — SECURITY & INTENT]
Security is central to Veritas AI. The backend uses a smart HTML sanitizer to scrub script tags and trackers, reducing token counts by 75%. Also, to prevent rate limit bottlenecks, we use local regex rules to triage emails, calling Gemini only for complex, ambiguous cases. Finally, every destructive action, like sending emails, has a strict human-in-the-loop gate requiring a final click before executing.
